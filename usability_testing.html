Usability Testing Report

I conducted usability testing on this portfolio website to ensure that users can easily navigate and understand the content, focusing on accessibility, clarity, and overall user experience. Testing involved 5 participants from diverse backgrounds, including both technical and non-technical users.

Key Findings:

  • Navigation: Users found the categorization intuitive but suggested adding breadcrumb trails or clearer back navigation for easier movement between sections.
  • Mobile Experience: Some users experienced layout issues on small screens, such as overlapping text or cramped menus.
  • Content Readability: While the content is detailed and meaningful, a few users recommended breaking up long paragraphs and adding more headings for scanning.
  • Load Performance: The site loaded quickly, though some CSS animations caused slight delays on older devices.
  • Visual Design: The minimalist style was appreciated for its clarity, but some users suggested a bit more contrast between links and text for better visibility.

Enhancements Made:

  • Separated the original single HTML file into multiple categorized pages to improve content organization and reduce page load times.
  • Improved mobile responsiveness using CSS media queries to adapt layouts, font sizes, and navigation for smaller screens.
  • Added clearer navigation aids including “Back to Home” links on all pages and grouped related content with logical menus.
  • Reformatted long text sections into smaller paragraphs with subheadings for easier reading and scanning.
  • Increased link color contrast for better accessibility and readability.
  • Simplified CSS animations to improve load performance on slower devices.

These enhancements aim to make the portfolio more user-friendly, accessible, and engaging for all viewers, whether prospective employers, peers, or mentors.
